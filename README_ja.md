# 10を作るゲームの答を列挙

[English](README.md)

## 概要

1から9までの整数を4つと四則演算、括弧のみで10を作る
回答を列挙する手抜きプログラム。

## Usage

```sh
make
./maketen
```

## 実行結果

```txt
(1+1)*(1+4)=10
(1+1)*(1*5)=10
((1+1)*1)*5=10
(1+(1*1))*5=10
((1+1)/1)*5=10
(1+(1/1))*5=10
(1+1)/(1/5)=10
    (中略)
9-((9/9)-2)=10
((9+9)/9)*5=10
(9+9)/(9/5)=10
((9+9)/9)+8=10
(9+(9*9))/9=10
((9*9)+9)/9=10
```
